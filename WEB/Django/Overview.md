# ✨ Django OVERVIEW ✨

## **Django**

- 빠르고 안전한 웹 애플리케이션 개발을 위한 고수준 파이썬 웹 프레임워크
- 주요 특징 : 신속한 개발, 보안 기능, 확장성, 유지보수 용이성, 강력한 ORM

## **install**

```bash
pip install django
```

- 설치 확인

```bash
django-admin
python -m django --version
```

## **기본 구조 생성**

```bash
django-admin startproject {project명} .
```
